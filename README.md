# ShadeRunner

A solo-developed, NSFW-forward, Shadowrun-1E-leaning CRPG anchored in Seattle 2050–2064.

- **Genre:** CRPG (BG3 / DOS2-like), isometric, turn-based, solo PC.
- **Setting:** Shadowrun 1E — Seattle (FASA, 1990). MVP district: **Redmond**.
- **Engine:** Godot 4, GDScript.
- **Platform:** Windows.
- **Policy:** personal project. NSFW fully unlocked. See `Design/Meta/DESIGN - NSFW.md`.

Layout:

- `Design/` — design decisions (Markdown). See `Design/DESIGN - Index.md`.
- `Data/` — canonical lore as YAML (per-district, per-faction, per-NPC, per-place).
- `Sources/` — original sourcebook page screenshots (cited from `Data/`).
- `Game/` — Godot project (code, scenes, shipped assets).
- `Media/` — concept art, references, raw art (pre-shipping).
