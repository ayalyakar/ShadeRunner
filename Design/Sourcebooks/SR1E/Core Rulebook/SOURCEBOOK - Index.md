# SOURCEBOOK - Index (SR1E Core Rulebook)

**Book:** Shadowrun First Edition — Core Rulebook (product code 7101)
**Publisher:** FASA Corporation
**Year:** 1989 (Corrected Second Printing)
**Designers:** Bob Charrette, Paul Hume, Tom Dowd
**Concept:** Jordan Weisman, L. Ross Babcock III, Dave Wylie, Bob Charrette,
Sam Lewis
**Tagline on cover:** *"Where man meets magic and machine."*

This index tracks chapter-by-chapter ingestion of the SR1E Core Rulebook. Each
sub-topic in the table below becomes its own file in this folder, named
`SOURCEBOOK - Title.md`, when ingested.

## Ingestion status legend

- **QUESTIONING** — listed but no content ingested yet.
- **DRAFT** — note written, actively under revision.
- **PARKED** — note good enough for now, will be revisited.
- **CONFLICT** — disagrees with another edition; timeline-aware rule applies.
- **BLOCKED** — waiting on another section's decisions.
