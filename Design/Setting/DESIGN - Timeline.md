# DESIGN - Timeline

## Decision

**DECIDED:**
- Game spans the full Shadowrun canonical timeline, **SR1E through SR6E** (~2050s → 2080s).
- The world **ages in sim-time**: characters age, factions evolve, canonical events occur.
- Timeline **branches** based on player (and world) actions beyond a small core of unstoppable events.

## Time model

- Start year is configurable at new-game; default proposals: 2050 (SR1), 2064 (SR3), 2070 (SR4), 2075 (SR5).
- Time advances continuously during play. See `World/DESIGN - Time.md` for granularity.
- Aging happens in-fiction; biologically the PC never dies from age (hidden mechanic) — see `Character/DESIGN - Aging.md`.
- When a protagonist arc closes, focus can shift to another PC; the world continues on the same timeline.

## Editions and mechanical transitions

Lore is strictly canonical; mechanics are a **custom hybrid across editions** (see `Systems/DESIGN - Combat.md`). Tech-era transitions surface in-world:

- **SR1–SR3 era:** cyberdecks, wired Matrix, jacked-in VR, pink-mohawk vibe; pay phones; SIN is rarer.
- **Crash 2.0 (2064):** Matrix destroyed, panic, world reshapes.
- **SR4 era:** wireless Matrix, commlinks, AR everywhere, technomancers emerge.
- **SR5 era:** hosts, grids, Public/Corporate, new decker economy.
- **SR6 era:** current canon, corp court politics, ongoing dragon civil war.

The player *experiences* these transitions as events in a run they started decades earlier.

## Transitions mechanically

Each edition transition:
- Fires scheduled canonical events in the event log.
- Promotes/demotes available gear and programs in the market.
- Rewrites some faction states (corp mergers, gang disappearances, new organizations).
- Can obsolete the player's gear or skills if they haven't kept up.

## Open questions

- Sim-time per real-time ratio during downtime — fast-forward options.
- Should the player be able to skip decades forward, or always live through them?
- How much in-world detail to render for eras the player didn't experience (news archives, flashbacks)?
- Do the 2010s "history" of the Awakening get any coverage, or is it exposition-only?
