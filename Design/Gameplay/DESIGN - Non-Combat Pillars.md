# DESIGN - Non-Combat Pillars

## Decision

**DECIDED:** Four non-combat pillars are all **first-class** — each can be the primary way a run is resolved, each has real mechanical depth:

1. **Stealth & infiltration**
2. **Social engineering & negotiation**
3. **Investigation & detective work**
4. **Heist planning & legwork**

A run can be won on any pillar without firing a shot. Combat is a choice, not a default.

## Stealth & infiltration

- Vision cones, sound propagation, lighting, line-of-sight break.
- Disguises tied to SIN tier and faction uniforms.
- Patrol patterns per guard schedule (procgen or authored).
- Distractions, takedowns, body hiding.
- Detection is graduated: suspicious → alerted → searching → combat.
- Skills and cyberware (Infiltration, Sneaking, chameleon suit, silencers, low-light) matter.

## Social engineering & negotiation

- Dialogue trees with skill checks (Etiquette, Con, Intimidation, Negotiation, Charisma).
- Dispositions of NPCs matter; relationships, current mood, lifestyle tier all factor.
- Negotiation mini-system for Johnson meets — haggle over pay, intel, conditions.
- Long cons: sustained impersonation across days/weeks with compounding risk.
- Cyberware (Social Adapter, Empathy Software) and spells (Influence) interact here.

## Investigation & detective work

- **Case-board metaphor** — physical/AR board with clues, photos, connections.
- **Evidence types**: documents, images, witness statements, forensic data, digital traces.
- **Interview mechanics** — Perception + Intuition reveal deception; body-language reads.
- **Deduction**: connecting clues unlocks next leads; dead ends require legwork.
- Magic can add (assensing auras, ritual scrying) and the Matrix can add (paydata digs).

## Heist planning & legwork

- **Legwork phase** — acquire blueprints, bribe guards, observe patterns, recruit specialists.
- **Planning phase** — select entry, route, tools, gear loadout, backup plans.
- **Run modifiers** — planning quality applies mechanical bonuses during execution (fewer surprises, better positions).
- **Specialists via contacts** — the PC can "hire" a decker or rigger for a run via Connection + Loyalty (see `Social/DESIGN - Contacts.md`).

## Cross-pillar interactions

- A successful investigation gives stealth bonuses (you know the patrol schedule).
- Strong social work replaces a hack (you get the password from a bribed janitor).
- Good legwork reduces combat difficulty (you know where the guards are).
- A run can (and should) combine pillars.

## Open questions

- Skill floors for each pillar — what's the minimum viable build for a stealth-only character?
- How to visualize "the plan" in UI so the player feels it pay off.
- AI detection tuning is notoriously brutal to balance.
- How much of investigation is puzzle vs. skill check vs. narrative?
